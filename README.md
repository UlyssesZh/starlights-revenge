# Starlight's Revenge

An interactive fiction.
[Play online.](https://ulysseszh.github.io/starlights-revenge)

Starlight Glimmer carries out a revenge after her leadership in her town is overthrown!

## Development

Use [Twine](https://twinery.org) to edit the HTML file.
Run `./build.rb` to build the web page.
